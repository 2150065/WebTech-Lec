JavaScript

Language used to specify the behavioral aspect (i.e., interactive effects) of a marked up document.

Externally linked
Can be placed anywhere and can be multiple
Embedded
Not recommended
Inline script
Event attributes (i.e., "on" + click)

JavaScript Data Types

    Boolean (true / false)
    Falsy - not exactly false but is equal to false.
    Truthy - not exactly true but is equal to true.
    Numbers
    Integers and decimals
    0b - binary types
    0 - octal notation
    0x - hexadecimal
    1e5 - exponential
    Always floating point division
    Strings
    "" , '' - delimit strings
    can be embedded - " ' ' "
    String interpolation - ` ` (evaluate)
    Undefined
    Null
    Standard Objects
    Mutator Methods
    Used to control changes to a variable
    • copyWithin()
    • fill()
    • pop()
    • push()
    • reverse()
    • shift()
    • sort()
    • splice()
    • unshift()
    Accessor Methods
    Do not affect or modify variables.
    Accessing methods from a collection one at a time.
    • concat()
    • indexOf()
    • join()
    • lastIndexOf()
    • slice()
    • toString()
    • toLocaleString()
    Iteration Methods
    • entries()
    • every()
    • filter()
    • find()
    • findIndex()
    • forEach()
    • keys()
    • map()
    • reduce()
    • reduceRight()
    • some()
    • values()
