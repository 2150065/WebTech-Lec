HTTP Module

  1. createClient() - Deprecated. Creates a HTTP client
  2. createServer() - Creates an HTTP server
  3. get() - Sets the method to GET, and returns an object containing the user's request
  4. globalAgent - Returns the HTTP Agent
  5. request() - Returns an object containing the user's request
  1. server.close() - Closes the server connection
  2. server.listen() - Makes the server listen to ports on the computer
  3. server.listening - Returns true if the server is listening for connection, otherwise false
  4. server.maxHeadersCount - Limits the number of incoming headers
  5. server.setTimeout() - Sets the server's timeout value. Default is 2 minutes
  6. server.timeout Sets, or gets, the server's timeout value
  1. checkContinue - Occurs when a request with an HTTP Expect: 100-continue is received
  2. checkExpectation - Occurs when a request with an HTTP Expect headers is received
  3. clientError - Occurs when a client connection gets an error
  4. close - Occurs when the server is closing
  5. connect - Occurs when a request with an HTTP Connect method is received connection
  6. request - Occurs when a request is received
  7. upgrade - Occurs when a request for an HTTP upgrade is received
  1. response.addTrailers() - Adds trailing headers at the end of the response
  2. response.end() - The response message is completed
  3. response.finished - True when the response has completed, otherwise false
  4. response.getHeader() - Returns the specified HTTP header
  5. response.headersSent - True if any headers has been sent, otherwise false
  6. response.removeHeader() - Removes the specified HTTP header
  7. response.sendDate - If set to true, a Date header will be sent with the response
  8. response.setHeader() - Sets the specified HTTP header
  9. response.setTimeout - Sets the maximum number of milliseconds of the response
  10. response.statusCode - Sets the HTTP status code of the response
  11. response.statusMessage - Sets the HTTP status message of the response
  12. response.write() - Sends the specified string content to the response body
  13. rresponse.writeContinue() - Sends a HTTP 100-Continue message
  14. response.writeHead() - Sends a response header
